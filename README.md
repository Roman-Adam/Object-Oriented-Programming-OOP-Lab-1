# Object-Oriented-Programming-OOP-Lab-1
Object-Oriented Programming (OOP) Lab 1 – Beginner-friendly C++ programs demonstrating the basics of classes, objects, member functions, arrays, and simple object-oriented concepts.

Object-Oriented Programming (OOP) – Lab 1

Overview

This repository contains the solutions for Lab 1 of the Object-Oriented Programming (OOP) course. The programs are implemented in C++ and are intended to help beginners understand the core concepts of object-oriented programming through the use of classes and objects.

Each program demonstrates a fundamental OOP concept, including class creation, data member definition, member function implementation, and object instantiation.

---

Objectives

- Understand the concepts of classes and objects.
- Learn how to define data members and member functions.
- Practice object creation and function invocation.
- Perform basic calculations using classes.
- Work with arrays of objects.

---

Lab Tasks

1. Student Class

- Create a "Student" class with the following members:
  - Name
  - Roll Number
  - Marks
- Input and display student information.

2. Rectangle Class

- Create a "Rectangle" class with the following members:
  - Length
  - Width
- Calculate and display the area.

3. BankAccount Class

- Create a "BankAccount" class with the following members:
  - Account Number
  - Balance
- Implement the following operations:
  - Deposit
  - Withdraw
  - Display Balance

4. Calculator Class

- Implement a simple calculator using a class.
- Perform the following operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division

5. Car Class

- Create a "Car" class with the following members:
  - Brand
  - Model
  - Price
- Display car details.

6. Person Class

- Create a "Person" class with the following members:
  - Name
  - Age
- Determine whether the person is eligible to vote.

7. Result Class

- Store the marks of five subjects.
- Calculate the percentage.
- Assign grades.
- Use an array of five objects to store student records.
